New here? How this tool works

This tool finds science questions where misinformation is actively circulating and generates evidence-based articles to combat it.

  1. Discovery — Every day, the pipeline scans science news, podcasts, Reddit questions, and Google Trends for claims that contradict established research. Those become candidate questions in the queue below.
  2. Review — An editor reviews the queue. Each question shows the misinformation narrative, why it matters, and what NASEM sources are available. Questions marked "needs review" haven't been editorially confirmed yet.
  3. Generate — Clicking "Generate Article" kicks off a 7-step pipeline that takes 3–5 minutes. It fetches and reads the source documents, extracts evidence, builds a consensus analysis, writes a draft at an 8th-grade reading level, then independently fact-checks every claim using a separate AI model. The result is a published article with a full evidence package you can inspect.
